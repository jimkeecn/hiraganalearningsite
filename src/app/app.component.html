Learning Hiragana

This is only a tool I have developed for myself learning hiragana. it allow me to remember the hiragana progressively. it's free to use for everyone

You need complete each test to progress. there will be cooldown between each test. it's not recommanded to skip cooldown. it allow time for you consume your knowledge.

Each test only has 1 hour to finish, you will need to start again if you run out of time.

This learing tool only work for if you use the same browser with cache on (not Incognito mode). If you clear your cache during the progress, you will lose all your progress and you have to start over.
